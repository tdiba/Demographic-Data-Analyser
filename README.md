### Project Overview
This Python project analyzes demographic data using Pandas to uncover insights about income, education, work patterns, and population distribution. The project was part of the freeCodeCamp Python certification.



### Project Objectives
The dataset was extracted from the 1994 Census database.The goal is to answer key questions about:
- Population distribution by race
- Education levels and income
- Work hours and earnings
- Country-level income comparisons
- Occupation trends



### Dataset
The dataset contains demographic information such as:
- Age
- Education
- Occupation
- Race
- Sex
- Hours worked per week
- Native country
- Salary (≤50K or >50K)



### Technologies Used
- Python
- Pandas
- Data Analysis
- Jupyter Notebook



### Questions Answered
1. How many people of each race are represented?
2. What is the average age of men?
3. What percentage of people have a Bachelor's degree?
4. What percentage of people with advanced education earn more than 50K?
5. What percentage of people without advanced education earn more than 50K?
6. What is the minimum number of hours worked per week?
7. What percentage of people working minimum hours earn more than 50K?
8. Which country has the highest percentage of people earning >50K?
9. What is the most popular occupation for high earners in India?


### Key Results
- Average age of men: 39.4
- Bachelor's degree holders: 16.4%
- Higher education earn >50K: 46.5%
- Lower education earn >50K: 17.4%
- Minimum work hours: 1 hour/week
- Highest earning country: Iran (41.9%)
- Top occupation in India (>50K): Prof-specialty
